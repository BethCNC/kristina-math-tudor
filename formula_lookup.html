Chapter 6: Personal Finance

Test 3 Material • Interest, Loans, Savings

Simple Interest

I = P × r × t

I = Interest, P = Principal, r = rate, t = time

Compound Interest

A = P(1 + r/n)^(nt)

A = Amount, P = Principal, r = rate, n = compounding periods per year, t = time

Monthly Payment (Loan)

PMT = P × [r(1+r)^n] / [(1+r)^n - 1]

PMT = Monthly payment, P = Principal, r = monthly rate, n = number of payments

Unit Conversion

New Value = Original Value × (Conversion Factor)

Multiply by conversion factor to change units

Area of Rectangle

A = l × w

A = Area, l = length, w = width
