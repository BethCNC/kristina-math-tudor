Key Formulas for Test 4

Basic Probability

P(A) = Number of favorable outcomes / Total outcomes

Probability is always between 0 and 1

Example: Rolling a 3 on a die: P(3) = 1/6 ≈ 0.167

Expected Value

E(X) = Σ [x × P(x)]

Sum of each outcome multiplied by its probability

Example: Expected value of rolling a die: (1×1/6) + (2×1/6) + ... + (6×1/6) = 3.5

Practice Problems for Test 4

Problem 1: Basic Probability

A bag contains 5 red marbles, 3 blue marbles, and 2 green marbles. What is the probability of drawing a red marble?

Solution:

Total marbles = 5 + 3 + 2 = 10

P(red) = 5/10 = 1/2 = 0.5

Problem 2: Expected Value

A game costs $2 to play. You win $5 if you roll a 6, otherwise you win nothing. What is the expected value?

Solution:

E(X) = ($5 × 1/6) + ($0 × 5/6) = $5/6 ≈ $0.83

Net expected value = $0.83 - $2.00 = -$1.17
