Essential Formulas

All the formulas you need for MAT 143, organized by topic

💰 Simple Interest

I = Prt

Variables:

  • I = Interest earned
  • P = Principal (starting amount)
  • r = Rate (as decimal)
  • t = Time (in years)

Example:

$1000 at 5% for 2 years: I = 1000 × 0.05 × 2 = $100

💰 Compound Interest

A = P(1 + r/n)^(nt)

Variables:

  • A = Final amount
  • P = Principal
  • r = Rate (as decimal)
  • n = Compounds per year
  • t = Time (years)

Example:

$1000 at 5% annually for 2 years: A = 1000(1.05)² = $1102.50

💰 APY (Annual Percentage Yield)

APY = ((1 + r/n)^n - 1) × 100%

Variables:

  • APY = Annual Percentage Yield
  • r = Nominal rate (as decimal)
  • n = Compounds per year

Example:

5% compounded monthly: APY = ((1 + 0.05/12)^12 - 1) × 100% = 5.12%

📏 Length Conversions

1 inch = 2.54 cm
1 foot ≈ 0.305 m

Key Conversions:

  • 1 yard = 3 feet = 0.914 m
  • 1 mile = 5280 feet = 1.609 km
  • 1 km = 1000 m = 0.621 miles

Example:

Convert 5 feet to meters: 5 × 0.305 = 1.525 m

🌡️ Temperature Conversion

F = 9C/5 + 32

Variables:

  • F = Fahrenheit
  • C = Celsius

Example:

Convert 20°C to Fahrenheit: F = 9(20)/5 + 32 = 68°F

📊 Z-Score

z = (x - μ)/σ

Variables:

  • z = Z-score
  • x = Data point
  • μ = Mean
  • σ = Standard deviation

Example:

If x=85, μ=80, σ=5: z = (85-80)/5 = 1

📊 Mean (Average)

μ = Σx/n

Variables:

  • μ = Mean
  • Σx = Sum of all values
  • n = Number of values

Example:

Values: 85, 90, 75, 95
Mean = (85+90+75+95)/4 = 86.25

⚖️ Proportions

a/b = c/d

Cross Multiplication:

  • ad = bc
  • a is to b as c is to d

Example:

If 3/2 = 9/x, then 3x = 18, so x = 6

💯 Percentages

Part = Whole × Rate
Rate = Part/Whole

Key Concepts:

  • Rate as decimal: 5% = 0.05
  • Rate as fraction: 5% = 5/100

Example:

15% of 200: Part = 200 × 0.15 = 30
