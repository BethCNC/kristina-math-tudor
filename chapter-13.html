Key Concepts

Voting Methods

  • Plurality Method: Candidate with most first-place votes wins
  • Borda Count: Points assigned based on ranking position
  • Instant Runoff: Eliminate lowest vote-getter, redistribute votes
  • Pairwise Comparison: Compare each pair of candidates head-to-head

Apportionment Formulas

Standard Divisor: SD = Total Population / Number of Seats

Standard Quota: SQ = State Population / SD

Practice Problems

Problem 1: Plurality Method

In an election with 3 candidates, A gets 45 votes, B gets 35 votes, C gets 20 votes. Who wins using plurality?

Solution:

Candidate A wins with 45 votes (most first-place votes)
