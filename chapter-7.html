Voting Methods

Chapter Overview

Explore different methods of voting and election systems! This chapter examines how democratic decisions are made, comparing various voting methods and their strengths and weaknesses.

Test 3 Coverage: This chapter will appear on Test 3 (Nov 3-7, 2025). Review all voting methods and practice problems!

Voting Methods

1. Plurality Method (First-Past-The-Post)

The candidate with the most votes wins. This is the most common method used in U.S. elections.

How it works:

  1. Each voter selects ONE candidate
  2. Count the total votes for each candidate
  3. The candidate with the most votes wins

Potential Issues:

  • • Winner may not have majority support (>50%)
  • • "Spoiler effect" - similar candidates can split votes
  • • Doesn't consider voter preferences beyond first choice

2. Borda Count Method

Voters rank all candidates. Points are awarded based on rank, and the candidate with the most points wins.

How it works:

  1. Voters rank ALL candidates (1st, 2nd, 3rd, etc.)
  2. Assign points: Last place gets 0 points, 2nd-to-last gets 1, etc.
  3. Sum all points for each candidate
  4. Highest total wins
Example: With 4 candidates (A, B, C, D):
• 1st place: 3 points
• 2nd place: 2 points
• 3rd place: 1 point
• 4th place: 0 points

3. Runoff Method (Sequential Runoff / Instant Runoff)

If no candidate has a majority, eliminate the candidate with fewest votes and redistribute their votes based on voters' next preferences.

How it works:

  1. Voters rank candidates by preference
  2. Count first-choice votes
  3. If no candidate has >50%, eliminate candidate with fewest votes
  4. Redistribute eliminated candidate's votes to voters' next choices
  5. Repeat until someone has majority

Advantage:

Winner always has majority support (>50%)

4. Pairwise Comparison (Copeland's Method)

Compare every candidate head-to-head. The candidate who wins the most one-on-one matchups wins.

How it works:

  1. Create all possible pairs of candidates (A vs B, A vs C, etc.)
  2. For each pair, count how many voters prefer one over the other
  3. Winner of each matchup gets 1 point
  4. Candidate with most matchup wins becomes overall winner
Note: With n candidates, you need n(n-1)/2 pairwise comparisons.
Example: 4 candidates = 4(3)/2 = 6 pairs to compare

Fairness Criteria

Majority Criterion

If a candidate receives >50% of first-place votes, they should win.

Condorcet Criterion

If a candidate would win every head-to-head matchup, they should win overall.

Monotonicity Criterion

If a candidate moves up in rankings, it shouldn't cause them to lose.

Independence of Irrelevant Alternatives (IIA)

Adding or removing a non-winning candidate shouldn't change the winner.

Arrow's Impossibility Theorem

Kenneth Arrow's theorem (1950s) proves that NO voting system can simultaneously satisfy all fairness criteria when there are 3+ candidates.

Key Insight:

Every voting method has flaws. The "best" method depends on what criteria you prioritize.

Practice Problems

Work through these problems to master voting methods:

  1. Given voter preference ballots, determine the winner using plurality method
  2. Calculate Borda Count points for each candidate and find the winner
  3. Apply instant runoff voting, eliminating candidates until one has majority
  4. Conduct pairwise comparisons and determine the Condorcet winner
  5. Identify which fairness criteria are violated in a given scenario
