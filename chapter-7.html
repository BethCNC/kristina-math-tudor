7.2 Dimensional Analysis

The unit cancellation method for conversions

The Goal: Make Units Cancel

Set up conversion factors as fractions so unwanted units cancel out

The key is putting the unit you want to cancel in both numerator and denominator

Step-by-Step Process
1 Write what you have
2 Set up conversion factor
3 Cancel matching units
4 Calculate the answer
Example: 5 feet to inches

Setup: 5 feet √ó (12 inches/1 foot)

Cancel: 5 ~~feet~~ √ó (12 inches/1 ~~foot~~)

Calculate: 5 √ó 12 = 60 inches

Answer: 60 inches
