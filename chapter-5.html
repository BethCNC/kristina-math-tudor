5.1 Function Basics

Understanding functions and function notation

What is a Function?

A function is a relationship where each input has exactly one output.

Think of it like a machine: you put something in, and you get exactly one thing out.

Function Notation

f(x) = 2x + 3

"f of x equals 2x plus 3"

f = name of the function

x = input value

f(x) = output value

Example

If f(x) = 2x + 3, find f(5)

Step 1: Replace x with 5

Step 2: f(5) = 2(5) + 3

Step 3: f(5) = 10 + 3 = 13

Answer: f(5) = 13

Practice Problems

Problem 1:

If f(x) = 3x - 7, find f(4).

Replace x with 4: f(4) = 3(4) - 7

Calculate: f(4) = 12 - 7

Answer: f(4) = 5

Problem 2:

A rental car company charges $30 per day plus $0.25 per mile. Write a linear function C(m) for the total cost based on miles driven m.

Identify parts: Base cost = $30, Cost per mile = $0.25

Linear function form: f(x) = mx + b

Set up function: C(m) = 0.25m + 30

Answer: C(m) = 0.25m + 30

5.2 Linear Functions

Understanding linear relationships and graphing lines

Linear Function Form

f(x) = mx + b

m = slope (how steep the line is)

b = y-intercept (where line crosses y-axis)

f(x) = y-value (output)

Understanding Slope
+ Positive slope: line goes up
− Negative slope: line goes down
0 Zero slope: horizontal line
Real-World Example

Phone Bill: f(x) = 0.10x + 25

x = minutes used

0.10 = cost per minute (slope)

25 = base monthly fee (y-intercept)

If you use 100 minutes: f(100) = $35

5.3 Exponential Functions

Understanding rapid growth and decay patterns

Exponential Function Form

f(x) = a(b)^x

a = initial value (starting amount)

b = growth factor

x = time periods

If b > 1: growth, if 0 < b < 1: decay

Growth Example

Population doubles every year

f(x) = 100(2)^x

Year 0: f(0) = 100

Year 1: f(1) = 200

Year 2: f(2) = 400

Decay Example

Car value loses 20% per year

f(x) = 20000(0.8)^x

Year 0: f(0) = $20,000

Year 1: f(1) = $16,000

Year 2: f(2) = $12,800

Quick Reference & Formulas

Key Formulas

  • f(x) = mx + b Linear function
  • f(x) = a(b)^x Exponential function
  • m = (y₂-y₁)/(x₂-x₁) Slope formula

Key Concepts

  • • Functions show input-output relationships
  • • Linear functions have constant rate of change
  • • Exponential functions have constant percent change
  • • Graphs help visualize function behavior
