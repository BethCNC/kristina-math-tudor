Linear Equations and Functions

Understanding linear relationships and the slope-intercept form

Key Concepts

Linear Function Form

y = mx + b

  • m = slope (rate of change)
  • b = y-intercept (starting value)
  • x = input variable
  • y = output variable
Slope Interpretation
  • ‚Ä¢ Positive slope: increasing
  • ‚Ä¢ Negative slope: decreasing
  • ‚Ä¢ Zero slope: horizontal
  • ‚Ä¢ Undefined slope: vertical
Slope Formula

m = (y‚ÇÇ - y‚ÇÅ)/(x‚ÇÇ - x‚ÇÅ)

Rise over run - change in y divided by change in x

Finding Slope Example

Points: (2, 5) and (6, 13)

Step 1: m = (13 - 5)/(6 - 2)

Step 2: m = 8/4 = 2

Slope: m = 2

Worked Examples

Example 1: Writing Linear Equations

A cell phone plan costs $30 per month plus $0.15 per text message. Write a linear function for the total monthly cost.

Step 1: Identify the components

  • ‚Ä¢ Fixed cost = $30 (y-intercept)
  • ‚Ä¢ Variable cost = $0.15 per text (slope)

Step 2: Set up the equation

C(t) = 0.15t + 30

Step 3: Interpret the function

  • ‚Ä¢ t = number of text messages
  • ‚Ä¢ C(t) = total monthly cost
  • ‚Ä¢ For 100 texts: C(100) = 0.15(100) + 30 = $45
Example 2: Finding Linear Equation from Two Points

Find the linear equation passing through points (1, 3) and (5, 11).

Step 1: Find the slope

m = (11 - 3)/(5 - 1) = 8/4 = 2

Step 2: Use point-slope form

y - y‚ÇÅ = m(x - x‚ÇÅ)

y - 3 = 2(x - 1)

Step 3: Convert to slope-intercept form

y - 3 = 2x - 2

y = 2x - 2 + 3

y = 2x + 1

Practice Problems

Problem 1:

A water tank contains 500 gallons and drains at 25 gallons per hour. Write a function for the water remaining after t hours.

Starting amount: 500 gallons

Rate of change: -25 gallons/hour (negative because draining)

Function: W(t) = -25t + 500

Check: After 10 hours: W(10) = -25(10) + 500 = 250 gallons

Problem 2:

Find the equation of the line passing through (3, 7) and (8, 22).

Find slope: m = (22 - 7)/(8 - 3) = 15/5 = 3

Point-slope form: y - 7 = 3(x - 3)

Simplify: y - 7 = 3x - 9

Final equation: y = 3x - 2

Comprehensive Example: Car Rental Company

A car rental company charges a base fee of $50 plus $0.30 per mile driven. Create a cost function and determine the cost for driving 200 miles.

Step 1: Identify components

  • ‚Ä¢ Base fee (fixed cost): $50
  • ‚Ä¢ Cost per mile (rate): $0.30
  • ‚Ä¢ Variable: miles driven (m)

Step 2: Set up function

C(m) = 0.30m + 50

Step 3: Calculate cost for 200 miles

C(200) = 0.30(200) + 50

C(200) = 60 + 50

C(200) = $110

Step 4: Interpret the result

It costs $110 to rent the car and drive 200 miles.

Exponential Functions

Understanding growth and decay with constant percent change

Exponential Function Form

f(x) = a(b)^x

  • a = initial value (when x = 0)
  • b = growth/decay factor
  • x = time or input variable
  • f(x) = output value
Factor Interpretation
  • ‚Ä¢ b > 1: exponential growth
  • ‚Ä¢ 0 < b < 1: exponential decay
  • ‚Ä¢ b = 1: constant (no change)
Exponential Growth

Growth factor: b = 1 + rate

Example: 5% growth -> b = 1.05

f(x) = 100(1.05)^x

Initial: $100, grows 5% per period

Exponential Decay

Decay factor: b = 1 - rate

Example: 20% decay -> b = 0.80

f(x) = 50000(0.80)^x

Initial: $50,000, loses 20% per period

Worked Examples

Example 1: Population Growth

A city's population is 50,000 and grows at 3% per year. Find the population after 10 years.

Step 1: Identify values

  • ‚Ä¢ Initial population: a = 50,000
  • ‚Ä¢ Growth rate: 3% per year
  • ‚Ä¢ Time: x = 10 years

Step 2: Find growth factor

b = 1 + 0.03 = 1.03

Step 3: Set up function

P(t) = 50,000(1.03)^t

Step 4: Calculate after 10 years

P(10) = 50,000(1.03)^10

P(10) = 50,000(1.344)

P(10) ‚âà 67,200 people

Example 2: Car Depreciation

A car worth $25,000 depreciates by 15% each year. Find its value after 5 years.

Step 1: Identify values

  • ‚Ä¢ Initial value: a = $25,000
  • ‚Ä¢ Depreciation rate: 15% per year
  • ‚Ä¢ Time: x = 5 years

Step 2: Find decay factor

b = 1 - 0.15 = 0.85

Step 3: Set up function

V(t) = 25,000(0.85)^t

Step 4: Calculate after 5 years

V(5) = 25,000(0.85)^5

V(5) = 25,000(0.444)

Example 3: Investment Growth

An investment of $1,000 grows at 7% annual interest, compounded annually. Find the value after 8 years.

Step 1: Identify the exponential model

  • ‚Ä¢ Principal: a = $1,000
  • ‚Ä¢ Interest rate: r = 7% = 0.07
  • ‚Ä¢ Time: t = 8 years

Step 2: Use compound interest formula

A(t) = 1000(1.07)^t

Step 3: Calculate

A(8) = 1000(1.07)^8

A(8) = 1000(1.718)

A(8) ‚âà $1,718

Practice Problems

Problem 1:

A bacteria culture starts with 200 bacteria and doubles every 4 hours. Write the exponential function and find the population after 12 hours.

Doubling means: growth factor b = 2 every 4 hours

Function: B(t) = 200(2)^(t/4), where t is in hours

After 12 hours: B(12) = 200(2)^(12/4) = 200(2)^3 = 200(8) = 1,600 bacteria

Problem 2:

The value of a collectible item decreases by 8% each year. If it's worth $500 today, what will it be worth in 6 years?

Decay rate: 8%, so factor b = 1 - 0.08 = 0.92

Function: V(t) = 500(0.92)^t

After 6 years: V(6) = 500(0.92)^6 = 500(0.606) ‚âà $303

Quick Reference & Formulas

Linear Functions

y = mx + b Slope-intercept form

Linear vs Exponential

Linear Functions
  • ‚Ä¢ Add/subtract same amount each time
  • ‚Ä¢ Constant rate of change
  • ‚Ä¢ Forms a straight line
  • ‚Ä¢ Example: Save $50 per month
Exponential Functions
  • ‚Ä¢ Multiply by same factor each time
  • ‚Ä¢ Constant percent change
  • ‚Ä¢ Forms a curved line
  • ‚Ä¢ Example: Population grows 2% per year
