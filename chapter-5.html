Functions & Growth Patterns

Discover the power of mathematical modeling! Learn how linear and exponential functions help us understand and predict real-world patterns from business growth to population changes.

Essential Formulas

Linear Functions

y = mx + b

m = slope (rate of change)

b = y-intercept (starting value)

Slope formula: m = (y₂-y₁)/(x₂-x₁)

Exponential Functions

y = a(b)ˣ

a = initial value

b = growth/decay factor

Growth: b > 1, Decay: 0 < b < 1

Tutorial: Linear Functions
What Makes a Function Linear?

Linear functions have a constant rate of change. This means for every step right, you move the same amount up or down.

Example: Your Job

You earn $15/hour → y = 15x

Every hour, you earn exactly $15 more

Visual Pattern

Points on a line: (1,15), (2,30), (3,45)

Always goes up by 15 for each +1 in x

Understanding Slope (m)

Slope = Rise/Run = Change in y / Change in x

Positive Slope

Line goes up → m > 0

Negative Slope

Line goes down → m < 0

Zero Slope

Flat line → m = 0

Understanding Y-Intercept (b)

The y-intercept is where your line crosses the y-axis (when x = 0). It's your starting value.

Real Example: Cell Phone Plan

y = 0.10x + 25

• $25 base fee (b = 25) - what you pay even with 0 minutes

• $0.10 per minute (m = 0.10) - how much each minute costs

Tutorial: Exponential Functions
What Makes a Function Exponential?

Exponential functions have a constant percent change. The amount of change gets bigger as the values get bigger.

Example: Population Growth

Population doubles every year

100 → 200 → 400 → 800

Growth Factor

Doubling = multiply by 2

So b = 2 in y = a(2)ˣ

Converting Percent to Growth Factor

Growth (Increase)

5% growth → b = 1 + 0.05 = 1.05

20% growth → b = 1 + 0.20 = 1.20

Decay (Decrease)

3% decay → b = 1 - 0.03 = 0.97

15% decay → b = 1 - 0.15 = 0.85

Memory Trick:

Growth factors > 1 make things bigger, factors < 1 make things smaller!

Practice Problems

Problem 1: Linear Function

A gym membership costs $30 per month plus a $50 enrollment fee. Write a linear function for total cost after x months.

Show Solution

Identify the parts:

• Monthly cost = $30 per month (this is the slope, m = 30)

• Enrollment fee = $50 one-time (this is the y-intercept, b = 50)

Formula: y = mx + b

Answer: y = 30x + 50

After 6 months: y = 30(6) + 50 = $230 total

Problem 2: Exponential Function

A bacteria population starts at 100 and triples every hour. Write an exponential function for population after x hours.

Show Solution

Identify the parts:

• Starting population = 100 (this is a = 100)

• Triples every hour = multiply by 3 (this is b = 3)

Formula: y = a(b)ˣ

Answer: y = 100(3)ˣ

After 3 hours: y = 100(3)³ = 100(27) = 2,700 bacteria

Problem 3: Finding Slope

Find the slope of the line passing through points (2, 5) and (6, 17).

Show Solution

Use slope formula: m = (y₂ - y₁)/(x₂ - x₁)

Points: (x₁, y₁) = (2, 5) and (x₂, y₂) = (6, 17)

Substitute: m = (17 - 5)/(6 - 2) = 12/4 = 3

Answer: The slope is 3

This means for every 1 unit right, the line goes up 3 units

Linear vs Exponential: Key Differences

Linear Functions

  • Constant rate of change (same amount each step)
  • Forms a straight line when graphed
  • Examples: hourly pay, phone bills, distance over time
  • Growth by addition: +5, +5, +5...

Exponential Functions

  • Constant percent change (same ratio each step)
  • Forms a curved line when graphed
  • Examples: population growth, compound interest, viral spread
  • Growth by multiplication: ×2, ×2, ×2...
