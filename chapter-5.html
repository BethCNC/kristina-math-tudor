Key Formulas

Linear Function

y = mx + b

m = slope, b = y-intercept

Example: y = 2x + 3 (slope = 2, y-intercept = 3)

Exponential Function

y = a × b^x

a = initial value, b = growth/decay factor, x = time

Example: y = 100 × 1.05^x (5% growth)

Practice Problems

Problem 1: Linear Function

Find the equation of a line that passes through points (2, 5) and (4, 11).

Solution:

Slope m = (11-5)/(4-2) = 6/2 = 3

Using point (2,5): 5 = 3(2) + b → b = -1

Equation: y = 3x - 1

Problem 2: Exponential Growth

A population of 1000 grows at 3% per year. What will it be after 5 years?

Solution:

y = 1000 × (1.03)^5

y = 1000 × 1.1593 ≈ 1,159
