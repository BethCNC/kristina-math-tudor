Key Formulas for Test 4

Mean (Average)

Mean = Σx / n

Sum of all values divided by number of values

Example: Data set: 5, 8, 10, 12, 15 → Mean = 50/5 = 10

Standard Deviation

σ = √[Σ(x - μ)² / n]

Measures spread of data around the mean

Note: Larger standard deviation = more spread out data

Z-Score

z = (x - μ) / σ

x = data value, μ = mean, σ = standard deviation

Example: If mean = 100, SD = 15, value = 115 → z = (115-100)/15 = 1.0

Practice Problems for Test 4

Problem 1: Calculate Mean

Find the mean of this data set: 12, 15, 18, 20, 25

Solution:

Mean = (12 + 15 + 18 + 20 + 25) / 5 = 90/5 = 18

Problem 2: Find Median

Find the median of this data set: 3, 7, 9, 15, 21, 24, 30

Solution:

Data already sorted. Middle value (4th of 7) = 15
